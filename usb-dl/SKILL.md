# USB 下载固件

## 功能
进入 `~/usb_dl` 目录，执行 `usb_dl` 工具将固件通过 USB 下载到 cv184x 开发板，并实时监看下载日志。

## 前置条件
- 终端中已设置环境变量 `SUDO_PASS`（sudo 密码）
- 开发板已通过 USB 连接到主机并处于下载模式
- `~/usb_dl/` 目录下存在 `usb_dl` 可执行文件和 `usb_dl_image/` 固件目录

## 执行步骤

严格按以下顺序在终端中执行：

### 第 1 步：进入工作目录
```bash
cd ~/usb_dl
```

### 第 2 步：执行 USB 下载
以后台方式启动下载命令，便于实时监看输出：
```bash
echo "$SUDO_PASS" | sudo -S ./usb_dl -c cv184x -i ./usb_dl_image/
```

### 第 3 步：监看日志
命令启动后，持续关注终端输出，重点监看以下内容：
- **下载进度**：关注百分比或 block 传输信息
- **错误信息**：如出现 `error`、`fail`、`timeout` 等关键字，立即报告给用户
- **完成标志**：出现 `success`、`done`、`complete` 等字样表示下载完成

## 异常处理
- 如果提示 **权限不足**：确认 `SUDO_PASS` 已正确设置
- 如果提示 **设备未找到**：提醒用户检查 USB 连接和开发板状态
- 如果出现 **超时**：建议用户重新让开发板进入下载模式后重试
- 如果环境变量 `SUDO_PASS` 未设置且默认密码无效，提示用户在终端中执行 `export SUDO_PASS="你的密码"` 进行设置

## 注意事项
- 必须先进入 `~/usb_dl` 目录再执行命令
- 下载过程中不要断开 USB 连接
- 如需更换固件目录，用户可指定替代路径替换 `./usb_dl_image/`
